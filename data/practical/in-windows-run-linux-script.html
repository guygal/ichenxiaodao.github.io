在Windows下执行Linux或Mac的脚本



两个工具介绍：
1、pscp.exe 从Windows拷贝文件到Linux;
2、plink.exe在Windows里面执行Linux的某个脚本。

格式 ：
pscp -r -C -l [login_user] -pw [login_pwd] [src_folder] [login_user]@[linux_addr]:[dest_folder]

pscp -r -C -l cdz -pw Abcd1234 D:\folder cdz@192.168.1.188:/opt/folder

plink -C -i cdz.private.ppk cdz@10.225.56.26 -pw Abcd1234 "/opt/script.sh"
