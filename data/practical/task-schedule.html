设置定时任务

Windows

（以Windows 7 为例）
Control Panel -> System and Security -> Schedule tasks -> Create New Task ...

关键点：
1、Triggers里面设置定时执行的周期；
2、Actions里面设置要执行的程序或脚本；

Mac OS

两个命令：
1、crontab -l 查看当前设置的定时任务；
2、crontab -e 编辑当前设置的定时任务。

举例定时任务格式：
0,30 9-22 * * 1-5 /User/cdz/task.sh
表示每周一到周五的9点到22点每隔半个小时执行脚本task.sh.
